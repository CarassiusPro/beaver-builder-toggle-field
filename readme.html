Using the Field

Basic Usage

'my_field' => array( 'type' => 'toggle', 'label' => __( 'Toggle Field', 'fl_builder' ), ),

Settings

You can change the labels from True and False, as well as the active colour

'toggle' => array( 'type' => 'toggle', 'label' => __( 'Toggle Field', 'fl_builder' ), 'default' => 'true', 'true' => 'Yes', 'false' => 'No'm 'color' => 'red', ),
