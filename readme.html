Using the Field

How to Load

To load the field, unzip to your plugin folder and then run the following function

function fp_add_toggle() { include( plugin_dir_path( __FILE__ ) . '/beaver-builder-toggle-field/toggle.php'); } add_action( 'init', 'fp_add_toggle' );

Basic Usage

'my_field' => array( 'type' => 'toggle', 'label' => __( 'Toggle Field', 'fl_builder' ), ),

Settings

You can change the labels and values from True and False, change the active and non active colour and have up to three buttons

'toggle' => array( 'type' => 'toggle', 'label' => __( 'Toggle Field', 'fl_builder' ), 'default' => 'no', 'options' => array( 'yes' => 'Yes', 'no' => 'No', 'maybe' => 'Maybe', ), 'oncolor' => 'red', 'offcolor' => '#d9edf7', ),
