Using the Field

Basic Usage

'my_field' => array( 'type' => 'toggle', 'label' => __( 'Toggle Field', 'fl_builder' ), ),

Settings

You can change the labels and values from True and False, as well as the active colour

'toggle' => array( 'type' => 'toggle', 'label' => __( 'Toggle Field', 'fl_builder' ), 'default' => 'true', 'options' => array( 'yes' => 'Yes' // yes becomes value and Yes becomes label instead of true 'no' => 'No' // no becomes value and No becomes label instead of false ), 'color' => 'red', ),
